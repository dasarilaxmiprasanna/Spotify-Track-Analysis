> 🎵 What Makes a Song Popular? — Spotify Track Analysis

This project analyzes Spotify tracks to explore how features like **danceability**, **energy**, **tempo**, and **valence** relate to a song's **popularity**. Built in **Python** using **Jupyter Notebook**, it focuses on **EDA**, **data cleaning**, and **visualization** with **Pandas**, **Seaborn**, **Matplotlib**, and **Plotly**.

> 🔍 Highlights
- Correlation heatmap of key features
- Interactive scatter plot (Plotly)
- Hit vs Flop comparison using boxplots
- Rule-of-thumb insights from averages
